To be able able to edit the ".di" format file, you need to install the Papyrus plugin for eclipse. 
This plugin is a UML tool that enables the user to create and edit UML diagrams.
The update site for the Papyrus plugin is the following:
Juno - http://download.eclipse.org/releases/juno
then type "papyrus" in the filter in order to have the list of all available tools for that release of Eclipse.
Other update sites are the following:
papyrus addons - http://www.papyrusuml.org/home/liblocal/docs/updates/papyrus
